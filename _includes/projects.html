Autonomous Line Following Robot using System Identification

Jan 2017 - May 2017
Academic Research Project,MAE 508/CHE 598 Digital Design and Control, ASU

• Worked on building an autonomous line following robot that could follow a coloured line. Used DFRobot Turtle with Arduino Uno and QTC-8RC Reflectance array sensor for line sensing and navigation

• Formulated the problem as a closed-loop system with Angular Position as the output and the input as a variable affecting both the actuators. The speed bias was kept as a constant for the motors

• Used Weighted Frequencies/Loop shaping methods to identify the robot's transfer function and then designed a controller for bandwidth and robustness.

Double Moon Classification problem using Multi Layer Perceptron

Aug 2017 - Dec 2017
Academic Research Project - EEE 511 Artificial Neural Computation

• Worked on creating a multilayer perceptron using artificial neural network techniques on the classic double moon classification problem for various distances between the moons.

• Used Multilayer Perceptron to solve the linear classification problem. Trained the data using Levenberg-Marquardt, Backpropagation, Backpropagation with momentum and solved the double moon classification for various values of distances between the half moons.

Time Series prediction using Artificial Neural Networks

Aug 2017 - Dec 2017
Academic Research Project - EEE 511 Artificial Neural Computation

• Forecasted future time data with limited data using a gradient descent algorithm in Neural networks using MATLAB.

• Used a perceptron with one hidden layer and multiple neurons to predict the data. Data was extracted from an excel sheet, then was normalized to values between (0,1) and then the narnet was created

• Data was trained using various algorithms including Levenberg-Marquardt, Bayessian Regularization and Back propagation with momentum.

Design & Modelling of a car cruise Control system using PC-104

Aug 2016 - Dec 2016

and hardware-in-the-loop

Academic Research Project - EEE 598 Feedback Systems Theory

• Obtained the differential equation of a car moving along a direction with opposing wind dynamics and gravity.
• Performed Hardware-in-Loop (HIL) testing using the PC104 and also via a secure server connection.

• Modelled and simulated the performance of the car with the help of Simulink and MATLAB for the linearized and Non-Linear models. Performed wind disturbance test for the speed control. Ongoing observations on stability.

• Ensured performance comparing the HIL results and checking for proper synchronization between hardware and software.
• Performed observability and controllability tests in MATLAB and checked for steady state outputs.

Master - Slave Robot (MS BOT): Thermal detection capable

Aug 2014 - May 2015

robots for human search operations

Academic Research Project - Bachelor's Thesis, Anna University

• Designed a stable and robust master slave robot configuration, fitted with thermal sensors using (LPC1700) ARM cortex M3 microprocessor with (SRF04) ultrasonic sensors for navigation to locate trapped humans in a disaster. Fitted the robot with cameras and GPS (PAM 7Q) to send the signal immediately to the control room in case of detection.

• Achieved almost 93% accuracy in trial runs with good detection ability and high speed of response.
